US配列ユーザー向け / ⌘英かな代替

左右⌘の単押しで
「かな / 英数」を切り替える

かなCmdは、US配列キーボード環境で左右の Command（⌘）キー単押しにより入力ソース（かな / 英数）を切り替える macOSメニューバー常駐アプリです。

⬇︎ Download（最新版） GitHubを見る

※ 初回起動でブロックされた場合：Finderで右クリック →「開く」

できること

左右⌘の単押しで切替

Commandキー単押しだけを検知し、かな/英数へ切り替えます。

割り当ての入れ替え

左⌘=かな／右⌘=英数 をワンタップで反転できます。

ログイン時起動

ログイン時に自動起動（macOS 13+）。メニューバーに常駐します。

インストール

1) Releases（latest） から zip をダウンロード
2) 展開して かなCmd.app を Applications（アプリケーション）へ移動
3) 権限を許可：入力監視（Input Monitoring） と アクセシビリティ（Accessibility）
4) うまく動かない場合はアプリ再起動／macOS再起動
