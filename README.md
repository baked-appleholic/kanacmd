# かなCmd（KanaCmd）

US配列キーボード環境で、左右の Command（⌘）キー単押しで「かな / 英数」を切り替える macOS メニューバーアプリです。  
「⌘英かな」がmacOSアップデートで不安定になったため、代替として作りました。

- 対象：USキーボード / US配列ユーザー
- 用途：Commandキー単押しで入力ソース切替（かな / 英数）

---

## 特徴
- 左⌘ / 右⌘ の単押しで入力ソース（かな / 英数）を切り替え
- 左右の割り当てを入れ替え可能
- Dock表示の切り替え
- ログイン時に自動起動（macOS 13+）
- 診断メニュー（権限・EventTap状態を表示）

---

## 対応OS
- macOS 13 以上（推奨：最新）
  - ログイン時起動（SMAppService）は macOS 13+ が必要

---

## インストール
1. Releases から `kanacmd.zip` をダウンロード
   [![Downloads](https://img.shields.io/github/downloads/crazyappleholic-oss/kanacmd/total)](https://github.com/baked-appleholic/kanacmd/releases)

3. 展開して `かなCmd.app` を **Applications（アプリケーション）** に移動
4. アプリを起動

> 初回起動時に「開発元を確認できません」等が出る場合：  
> Finderでアプリを右クリック → **開く** → もう一度 **開く**

---

## 必要な権限（重要）
本アプリはキーボード入力を監視して入力切替を行うため、以下の権限が必要です。

### 1) 入力監視（Input Monitoring）
- システム設定 → プライバシーとセキュリティ → **入力監視**
- `かなCmd` をON

### 2) アクセシビリティ（Accessibility）
- システム設定 → プライバシーとセキュリティ → **アクセシビリティ**
- `かなCmd` をON

※アプリ内メニュー **「診断：状態を見る」** で確認できます。

---

## 使い方
- 左⌘単押し：かな（または英数）
- 右⌘単押し：英数（またはかな）

割り当てはメニューから変更できます：
- 「左右の割り当てを入れ替え」

---

## うまく動かないとき（トラブルシュート）
- 「診断：状態を見る」で権限が **未許可** になっていないか確認
- 一度アプリを終了して再起動
- システム設定で `かなCmd` をOFF→ONし直す
- Xcodeからの実行と /Applications からの実行が混ざっていないか確認（権限が別扱いになることがあります）
